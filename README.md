# ASL Dictionary App

PWA with ASL signs and Firebase Auth.

## Setup
1. Firebase project: Enable Auth (Google/FB/Twitter).
2. Replace firebaseConfig in script.js.
3. Open index.html.

## Features
- Auth overlay (social/email, remember/forget/skip).
- 16 signs with GIFs.
- Categories, search, offline PWA.
